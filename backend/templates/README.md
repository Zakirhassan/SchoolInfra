# PDF Templates

This directory contains PDF templates for certificates and ID cards.

## Required Templates

1. **certificate_template.pdf** - Certificate template with placeholders for:
   - Student name
   - Course name
   - Start date
   - End date
   - Grade
   - Student ID
   - Academic year

2. **idcard_template.pdf** - ID card template with placeholders for:
   - Student name
   - Father's name
   - Course name
   - Batch number
   - Date of birth
   - Contact number
   - Photo (optional)

## Template Coordinates

The coordinates for text placement are configured in `pdfGenerator.js`.
You can adjust these coordinates to match your template layout.

## How to Update Templates

1. Place your PDF template files in this directory
2. Update the coordinate mappings in `pdfGenerator.js`
3. Restart the backend server
